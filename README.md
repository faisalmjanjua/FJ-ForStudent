### VanillaJS

* [Calculator-js] - HTML enhanced for web apps!

#### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ gulp watch
```
 
### Angular

* [Calculator-ng] - HTML enhanced for web apps!

#### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ ng serve --o
```
